## StudyWise
A simple study session tracker

<b>Tech Stack</b>: </br>
Tailwind </br>
Lucide React Icons </br>
NextJS </br>
Local Storage <i>(for now)</i>

## Getting Started

Run the development server:

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
